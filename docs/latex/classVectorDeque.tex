\subsection{Detailed Description}
\subsubsection*{template$<$class Data\-Type$>$class Vector\-Deque$<$ Data\-Type $>$}

{\ttfamily \hyperlink{classVectorDeque}{Vector\-Deque}} satisfies the resource constraints typically expected of both Vectors and Deques. 

In particular, it has


\begin{DoxyItemize}
\item {\ttfamily O(1)} Member access
\item {\ttfamily O(1)} Append
\item {\ttfamily O(1)} Prepend
\end{DoxyItemize}

Additionally, {\ttfamily \hyperlink{classVectorDeque}{Vector\-Deque}} will only automatically resize when it is at full capacity. Iterators for {\ttfamily \hyperlink{classVectorDeque}{Vector\-Deque}} maintain an index to the element they are currently pointing to, and modifications to the {\ttfamily \hyperlink{classVectorDeque}{Vector\-Deque}} will not change that index. For example, suppose an element is added to the front of the {\ttfamily \hyperlink{classVectorDeque}{Vector\-Deque}} while an iterator is pointing to the third element. That iterator will now be pointing to what was previously the second element, since that element is now the third element and the iterator's index did not change.
